Arduino Remote

This is a simple demo of a set of technologies we can use to connect to an Arduino over the internet.

GitHub

This project is hosted on GitHub.

Components

  • Arduino: The Arduino itself runs the code 'controller.ino', and has circuits attached to it.
  • Client: The client runs on a local machine, and is connected to the Arduino with a USB cable.
  • Server: The server runs on a publicly available server.

Events & Actions

  • An Event is something that happens on the Arduino, and is sent to the server.
  • An Action is a command that is sent to the server, which causes something to happen on the Arduino.

Checking for events

You can check for events with a GET request to /events.

Sending actions

Send actions to the arduino with a POST request to /actions.

Example actions

Turn on the green LED.

POST /actions {'type': 'led', 'target': 'led1', 'name': '', 'value': 'on'}

Turn the RGB LED "red" (try "green" and "blue").

POST /actions {'type': 'led', 'target': 'rgb_led', 'name': 'red', 'value': '100'}

Play music

POST /actions {'type': 'piezo', 'target': '', 'name': '', 'value': 'edcdeeedddeeeedcdeeeeddedc'}
